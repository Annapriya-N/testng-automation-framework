# testng-automation-framework
A Java-based TestNG automation framework demonstrating annotation usage, XML suite configuration, and method inclusion/exclusion logic. Includes examples of @BeforeSuite, @BeforeClass, and grouped test execution for structured test management.

# TestNG Automation Framework

This repository contains a sample TestNG project built in Java to illustrate key TestNG concepts:
- Test annotations (`@BeforeSuite`, `@BeforeClass`, `@Test`)
- XML suite configuration with method and group filtering
- Parallel execution using `thread-count` and `parallel` attributes
- Organized package structure (`framework`)

## Features
- Demonstrates TestNG lifecycle and grouping
- Shows how to exclude or include test methods via XML
- Ready for integration with Selenium WebDriver for UI automation

## How to Run
1. Clone the repository.
2. Import into Eclipse or IntelliJ.
3. Run the `testng.xml` suite file.
